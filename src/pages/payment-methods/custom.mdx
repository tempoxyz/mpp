# Custom [Build your own payment method]

The mpay SDK is designed for extensibility. You can implement custom payment methods to integrate any payment rails—Lightning Network, other blockchains, card processors, or proprietary systems.

## Overview

A custom payment method requires:

1. **Intent definitions** — Schema for request parameters and credential payloads
2. **Method definition** — Combines intents under a named method
3. **Client logic** — Creates credentials when encountering 402 responses
4. **Server logic** — Verifies credentials and returns receipts

## Define an intent

Start by defining the schema for your payment intent. The schema specifies what parameters the server requests and what the client must provide.

```ts twoslash
import { Intent, z } from 'mpay'

const charge = Intent.from({
  name: 'charge',
  schema: {
    request: z.object({
      amount: z.string(),
      currency: z.string(),
      recipient: z.string(),
    }),
  },
})
```

## Attach method-specific schemas

Extend the intent with method-specific request fields and credential payload schemas. This example uses Lightning Network:

```ts twoslash
import { Intent, MethodIntent, z } from 'mpay'

const charge = Intent.from({
  name: 'charge',
  schema: {
    request: z.object({
      amount: z.string(),
      currency: z.string(),
      recipient: z.string(),
    }),
  },
})
// ---cut---
const lightningCharge = MethodIntent.fromIntent(charge, {
  method: 'lightning',
  schema: {
    request: {
      methodDetails: z.object({
        invoice: z.string(),
        paymentHash: z.string(),
      }),
    },
    credential: {
      payload: z.object({
        preimage: z.string(),
      }),
    },
  },
})
```

## Create the method

Compose your intents into a method definition:

```ts twoslash
import { Intent, Method, MethodIntent, z } from 'mpay'

const charge = Intent.from({
  name: 'charge',
  schema: {
    request: z.object({
      amount: z.string(),
      currency: z.string(),
      recipient: z.string(),
    }),
  },
})

const lightningCharge = MethodIntent.fromIntent(charge, {
  method: 'lightning',
  schema: {
    request: {
      methodDetails: z.object({
        invoice: z.string(),
        paymentHash: z.string(),
      }),
    },
    credential: {
      payload: z.object({
        preimage: z.string(),
      }),
    },
  },
})
// ---cut---
const lightningMethod = Method.from({
  name: 'lightning',
  intents: { charge: lightningCharge },
})
```

## Client implementation

Extend the method with credential creation logic. This runs when the client encounters a 402 response:

```ts twoslash
import { Credential, Intent, Method, MethodIntent, z } from 'mpay'

const charge = Intent.from({
  name: 'charge',
  schema: {
    request: z.object({
      amount: z.string(),
      currency: z.string(),
      recipient: z.string(),
    }),
  },
})

const lightningCharge = MethodIntent.fromIntent(charge, {
  method: 'lightning',
  schema: {
    request: {
      methodDetails: z.object({
        invoice: z.string(),
        paymentHash: z.string(),
      }),
    },
    credential: {
      payload: z.object({
        preimage: z.string(),
      }),
    },
  },
})

const lightningMethod = Method.from({
  name: 'lightning',
  intents: { charge: lightningCharge },
})

declare function payInvoice(invoice: string): Promise<{ preimage: string }>
// ---cut---
const clientMethod = Method.toClient(lightningMethod, {
  async createCredential({ challenge }) {
    // Pay the BOLT11 invoice via Lightning node
    const bolt11 = challenge.request.methodDetails!.invoice;
    const result = await payInvoice(bolt11)

    return Credential.serialize({
      challenge,
      payload: {
        preimage: result.preimage,
      },
    })
  },
})
```

## Server implementation

Extend the method with verification logic. For Lightning, verify the preimage hashes to the payment hash:

```ts twoslash
import { Intent, Method, MethodIntent, Receipt, z } from 'mpay'

const charge = Intent.from({
  name: 'charge',
  schema: {
    request: z.object({
      amount: z.string(),
      currency: z.string(),
      recipient: z.string(),
    }),
  },
})

const lightningCharge = MethodIntent.fromIntent(charge, {
  method: 'lightning',
  schema: {
    request: {
      methodDetails: z.object({
        invoice: z.string(),
        paymentHash: z.string(),
      }),
    },
    credential: {
      payload: z.object({
        preimage: z.string(),
      }),
    },
  },
})

const lightningMethod = Method.from({
  name: 'lightning',
  intents: { charge: lightningCharge },
})

declare function sha256(data: Uint8Array): Uint8Array
declare function hexToBytes(hex: string): Uint8Array
declare function bytesToHex(bytes: Uint8Array): string
// ---cut---
const serverMethod = Method.toServer(lightningMethod, {
  async verify({ credential, request }) {
    const preimage = credential.payload.preimage
    const expectedHash = credential.challenge.request.methodDetails!.paymentHash

    // Verify preimage: SHA256(preimage) === paymentHash
    const actualHash = bytesToHex(sha256(hexToBytes(preimage)))
    const isValid = actualHash === expectedHash

    return Receipt.from({
      method: 'lightning',
      status: 'success',
      reference: preimage,
      timestamp: new Date().toISOString(),
    })
  },
})
```

## Use in your app

### Client

Pass the client method to `Fetch.from`:

```ts twoslash
import { Credential, Intent, Method, MethodIntent, z } from 'mpay'
import { Fetch } from 'mpay/client'

const charge = Intent.from({
  name: 'charge',
  schema: {
    request: z.object({
      amount: z.string(),
      currency: z.string(),
      recipient: z.string(),
    }),
  },
})

const lightningCharge = MethodIntent.fromIntent(charge, {
  method: 'lightning',
  schema: {
    request: {
      methodDetails: z.object({
        invoice: z.string(),
        paymentHash: z.string(),
      }),
    },
    credential: {
      payload: z.object({
        preimage: z.string(),
      }),
    },
  },
})

const lightningMethod = Method.from({
  name: 'lightning',
  intents: { charge: lightningCharge },
})

declare function payInvoice(invoice: string): Promise<{ preimage: string }>

const clientMethod = Method.toClient(lightningMethod, {
  async createCredential({ challenge }) {
    const bolt11 = challenge.request.methodDetails!.invoice;

    return Credential.serialize({

})
// ---cut---
const fetch = Fetch.from({
  methods: [clientMethod],
})

const response = await fetch('https://api.example.com/premium')
```

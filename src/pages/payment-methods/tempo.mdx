# Tempo [Instant stablecoin payments on the Tempo blockchain]

The [Tempo](https://docs.tempo.xyz) payment method enables instant payments using TIP-20 stablecoins on the Tempo blockchain.

## Overview

[Tempo](https://docs.tempo.xyz) is purpose-built for machine-to-machine payments. It provides:

- **Instant finality:** Transactions settle in ~500ms with deterministic confirmation
- **Low fees:** Sub-cent transaction costs suitable for micropayments
- **Fee sponsorship:** Servers can optionally cover gas fees for clients

## Intents

:::info[Advanced intents]
Future versions will support additional intents like streaming payments for metered usage.
:::

### Charge

The `tempo` method supports the `charge` intent for immediate one-time payments.

#### Client

Use the `tempo` method from `mpay/client` to automatically sign payment transactions when encountering 402 responses.

```ts twoslash
import { Fetch, tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const fetch = Fetch.from({
  methods: [
    tempo({
      account: privateKeyToAccount('0x...'),
      rpcUrl: 'https://rpc.tempo.xyz',
    }),
  ],
})

const response = await fetch('https://mpp.dev/ping/paid')
```

See [`tempo` client reference](/sdk/typescript/client/Method.tempo) for full API documentation.

#### Server

Use the `tempo` method from `mpay/server` to verify payment credentials and settle transactions.

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  method: tempo({
    chainId: 42431,
    rpcUrl: 'https://rpc.tempo.xyz',
  }),
  realm: 'mpp.dev',
  secretKey: process.env.MPAY_SECRET_KEY!,
})

export async function handler(request: Request) {
  // [!code focus:start]
  const result = await mpay.charge({
    request: {
      amount: '1000000',
      currency: '0x20c0000000000000000000000000000000000001',
      expires: '2030-01-20T12:00:00Z',
      recipient: '0xa726a1CD723409074DF9108A2187cfA19899aCF8',
    },
  })(request)
  // [!code focus:end]

  if (result.status === 402) return result.challenge

  return result.withReceipt(Response.json({ data: '...' }))
}
```

See [`tempo.charge` server reference](/sdk/typescript/server/Method.tempo.charge) for full API documentation.

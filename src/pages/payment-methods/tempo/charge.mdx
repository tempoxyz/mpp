# Charge [Immediate one-time payments]

The `charge` intent requests an immediate one-time payment from the client.

<Tabs stateKey="platform">
  <Tab title="Server">
  <div className="vocs:space-y-4">
Use the [`.charge` method](/sdk/typescript/server/Method.tempo.charge) on your [`Mpay` instance](/sdk/typescript/server/Mpay.create) to verify and settle payments when a user accesses your endpoint.

```ts twoslash
// @noErrors
import { Mpay, tempo } from 'mpay/server' // [!code collapse:8]

const mpay = Mpay.create({
  method: tempo(),
})

export async function handler(request: Request) {
  // [!code focus:start]
  const result = await mpay.charge({
    amount: '1000000',
    currency: '0x20c0000000000000000000000000000000000001',
    recipient: '0xa726a1CD723409074DF9108A2187cfA19899aCF8',
  })(request)
  // [!code focus:end]

  if (result.status === 402) return result.challenge

  return result.withReceipt(Response.json({ data: '...' }))
}
```

:::info
See [`tempo.charge` server reference](/sdk/typescript/server/Method.tempo.charge) for full API documentation.
:::
  </div>
  </Tab>
  <Tab title="Client">
  <div className="vocs:space-y-4">
Use the [`tempo` method](/sdk/typescript/client/Method.tempo) on a [`Fetch` instance](/sdk/typescript/client/Fetch.from) to automatically sign payment transactions when encountering 402 responses.

```ts twoslash
import { Fetch, tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const account = privateKeyToAccount('0x...')

const fetch = Fetch.from({
  methods: [tempo({ account })],
})

const response = await fetch('https://mpp.dev/ping/paid')
```

:::info
See [`tempo` client reference](/sdk/typescript/client/Method.tempo) for full API documentation.
:::

  </Tab>

# Tempo [Instant stablecoin payments on the Tempo blockchain]

The [Tempo](https://docs.tempo.xyz) payment method enables instant payments using TIP-20 stablecoins on the Tempo blockchain.

## Overview

[Tempo](https://docs.tempo.xyz) is purpose-built for machine-to-machine payments. It provides:

- **Instant finality:** Transactions settle in ~500ms with deterministic confirmation
- **Low fees:** Sub-cent transaction costs suitable for micropayments
- **Fee sponsorship:** Servers can optionally cover gas fees for clients

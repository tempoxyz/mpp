# Services [MPP-enabled APIs your agent can pay for]

These services accept MPP payments today. Your agent pays per request with stablecoins—no API keys, no accounts, no manual setup.

Every endpoint below is a proxy that handles payment via the MPP `402` Challenge-Credential-Receipt flow. Point your agent at any of them and it pays automatically through [Presto](https://github.com/tempoxyz/presto).

## Get started

Install Presto, connect a wallet, and your agent handles the rest.

<AgentTabs />

The **Connect wallet** step (`presto login`) is the only one that needs you—complete it once in your browser. After that, your agent uses Presto for payments automatically. For a full reference you can give your agent, see [setup.md](/setup.md).

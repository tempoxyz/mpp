# Services [Use MPP-enabled APIs your agent or application can pay for]

Discover MPP services that are now live, and how to use them via the examples below. Once you've funded a supported payment method, you can begin using MPP services from your agent or application for seamless, real-time, and programmable payments.

:::tip
Examples use [Tempo](/payment-methods/tempo) for payments, where [stablecoins](/payment-methods/tempo#stablecoins) enable low cost, instant, and secure payments. Services could also offer [Stripe](/payment-methods/stripe), with more payment methods on the way.
:::

## Setup

### ⚙️ Developers

Try using a MPP-enabled service from an application or server with the `mppx` SDKs, which interact with the Tempo network. These SDKs are available in [TypeScript](/sdk/typescript), [Python](/sdk/python), and [Rust](/sdk/rust).

### 🤖 Agents

Supercharge your agent with paid services. Use [presto](/tools/presto) to create or sign into a wallet, and deposit funds via supported methods to start leveraging MPP with your agent.

<AgentTabs />

:::note
When using `presto` to sign in or create a Tempo account, your browser will prompt to save or use your passkey — no passwords required. From the [dashboard](https://app.tempo.xyz), you can add funds via Apple Pay, credit card, or crypto at any time.
:::

You can use the MPP-enabled services below via the Tempo network. More services and supported payment methods are added regularly. To add MPP support to your API, start integrating [here](/quickstart/server).

# For humans [MPP lets AI agents autonomously pay for services—no API keys, no billing dashboards, no manual signup.]

## Use with an agent

Already using Claude, Codex, or another AI agent? Just run this command:

<AgentTabs />

The agent will set itself up with `tempoctl` and start using paid services automatically.

## Integrate into your app

Building an application that needs to make paid API calls? Use the SDK.

::::steps

### Install

:::code-group

```bash [npm]
npm install mpay viem
```

```bash [pnpm]
pnpm add mpay viem
```

```bash [bun]
bun add mpay viem
```

:::

### Set up payments

```ts twoslash
import { privateKeyToAccount } from "viem/accounts";
import { Mpay, tempo } from "mpay/client";

const account = privateKeyToAccount("0xabc…123");

Mpay.create({
  methods: [tempo({ account })],
});
```

### Use fetch as normal

```ts
const response = await fetch(
  "https://openrouter.payments.tempo.xyz/v1/chat/completions",
  {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      model: "anthropic/claude-3.5-sonnet",
      messages: [{ role: "user", content: "Hello!" }],
    }),
  },
);
```

Payments happen automatically when services return `402`.

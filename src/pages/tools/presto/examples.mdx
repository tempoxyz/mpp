# presto examples [Test against live endpoints]

## Try it now

We host a live demo server at `mpp.tempo.xyz` running on **Tempo testnet** that you can test against:

```bash [try-demo.sh]
# Free endpoint - no payment required
$ presto https://mpp.tempo.xyz/api/ping

# Paid endpoint - requires 0.10 pathUSD payment
$ presto https://mpp.tempo.xyz/api/ping/paid

# Dry run to preview what happens
$ presto --dry-run https://mpp.tempo.xyz/api/ping/paid

# Verbose output to see the full 402 flow
$ presto -vi https://mpp.tempo.xyz/api/ping/paid
```

:::tip[Get testnet funds]
The demo server runs on Tempo testnet. Fund your wallet at [faucet.tempo.xyz](https://faucet.tempo.xyz) with pathUSD before testing.
:::

## Basic requests

```bash [basic-requests.sh]
# Make a payment request
$ presto https://mpp.tempo.xyz/api/ping/paid

# Verbose output with headers
$ presto -vi https://mpp.tempo.xyz/api/ping/paid

# Save output to file
$ presto -o output.json https://mpp.tempo.xyz/api/ping/paid

# JSON output format
$ presto --output-format json https://mpp.tempo.xyz/api/ping/paid
```

## Payment controls

```bash [payment-controls.sh]
# Preview payment without executing (dry run)
$ presto --dry-run https://mpp.tempo.xyz/api/ping/paid
```

## Session management

```bash [session-management.sh]
# List active payment sessions
$ presto session list

# Close a specific session
$ presto session close https://mpp.tempo.xyz

# Close all sessions
$ presto session close --all

# Recover a session from on-chain state
$ presto session recover https://mpp.tempo.xyz
```

## Wallet management

```bash [wallet-management.sh]
# Show wallet address and balances
$ presto whoami

# Same as whoami
$ presto balance

# JSON output
$ presto whoami --output-format json
```

## HTTP options

```bash [http-options.sh]
# Custom headers
$ presto -H "X-Custom: value" https://mpp.tempo.xyz/api/ping/paid

# POST with data
$ presto -d '{"key": "value"}' https://mpp.tempo.xyz/api/ping/paid

# POST with JSON (sets Content-Type automatically)
$ presto --json '{"key": "value"}' https://mpp.tempo.xyz/api/ping/paid

# POST data from file
$ presto -d @data.json https://mpp.tempo.xyz/api/ping/paid

# Disable following redirects
$ presto --no-redirect https://mpp.tempo.xyz/api/ping/paid

# Set timeout
$ presto -m 30 https://mpp.tempo.xyz/api/ping/paid
```

## Output control

```bash [output-control.sh]
# Quiet mode (suppress non-essential output)
$ presto -q https://mpp.tempo.xyz/api/ping/paid

# Include HTTP headers in output
$ presto -i https://mpp.tempo.xyz/api/ping/paid

# Disable colors
$ presto --color never https://mpp.tempo.xyz/api/ping/paid
```

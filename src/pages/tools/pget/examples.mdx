# pget examples [Test against live endpoints]

## Try it now

We host a live demo server at `mpp.tempo.xyz` running on **Tempo testnet** that you can test against:

```bash [try-demo.sh]
# Free endpoint - no payment required
$ pget https://mpp.tempo.xyz/ping

# Paid endpoint - requires 0.10 pathUSD payment
$ pget https://mpp.tempo.xyz/ping/paid

# Inspect the payment requirements first
$ pget inspect https://mpp.tempo.xyz/ping/paid

# Dry run to preview what happens
$ pget -D https://mpp.tempo.xyz/ping/paid

# Verbose output to see the full 402 flow
$ pget -vi https://mpp.tempo.xyz/ping/paid
```

### Set up with a private key

For testing, you can use a private key directly using an environment variable:

```bash [set-key.sh]
# Set your private key (funded on Tempo testnet)
$ export PGET_PRIVATE_KEY="0x..."

# Make paid requests
$ pget https://mpp.tempo.xyz/ping/paid
```

:::tip[Get testnet funds]
The demo server runs on Tempo testnet. Fund your wallet at [faucet.tempo.xyz](https://faucet.tempo.xyz) with pathUSD before testing.
:::

## Basic requests

```bash [basic-requests.sh]
# Make a payment request
$ pget https://mpp.tempo.xyz/ping/paid

# Verbose output with headers
$ pget -vi https://mpp.tempo.xyz/ping/paid

# Save output to file
$ pget -o output.json https://mpp.tempo.xyz/ping/paid

# JSON output format
$ pget --json-output https://mpp.tempo.xyz/ping/paid
```

## Payment controls

```bash [payment-controls.sh]
# Preview payment without executing (dry run)
$ pget -D https://mpp.tempo.xyz/ping/paid

# Require confirmation before payment
$ pget -y https://mpp.tempo.xyz/ping/paid

# Set maximum payment amount (in atomic units)
$ pget -M 10000 https://mpp.tempo.xyz/ping/paid

# Filter to specific networks
$ pget -n tempo-moderato https://mpp.tempo.xyz/ping/paid
```

## Wallet management

```bash [wallet-management.sh]
# Check wallet balance
$ pget balance

# Check balance on specific network
$ pget balance -n tempo-moderato

# Use specific account by name
$ pget -a my-wallet https://mpp.tempo.xyz/ping/paid

# Use specific sender address
$ pget --from 0x1234... https://mpp.tempo.xyz/ping/paid
```

## Inspecting payments

```bash [inspect-payments.sh]
# Inspect payment requirements without paying
$ pget inspect https://mpp.tempo.xyz/ping/paid

# View configuration
$ pget config

# View config as JSON
$ pget config --output-format json
```

## HTTP options

```bash [http-options.sh]
# Custom headers
$ pget -H "X-Custom: value" https://mpp.tempo.xyz/ping/paid

# POST with data
$ pget -X POST -d '{"key": "value"}' https://mpp.tempo.xyz/ping/paid

# POST with JSON (sets Content-Type automatically)
$ pget --json '{"key": "value"}' https://mpp.tempo.xyz/ping/paid

# Follow redirects
$ pget -L https://mpp.tempo.xyz/ping/paid

# Set timeout
$ pget -m 30 https://mpp.tempo.xyz/ping/paid
```

## Output control

```bash [output-control.sh]
# Quiet mode (suppress non-essential output)
$ pget -q https://mpp.tempo.xyz/ping/paid

# Show only HTTP headers
$ pget -I https://mpp.tempo.xyz/ping/paid

# Disable colors
$ pget --color never https://mpp.tempo.xyz/ping/paid
```

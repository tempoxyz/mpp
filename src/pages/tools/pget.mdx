# pget [wget for paid APIs]

`pget` is a non-interactive commandline tool for making HTTP requests with automatic payment support.

When you request a resource that requires payment, `pget` automatically detects the `402` response, fulfills the payment, and retries—all in a single command.

## Installation

**Quick install:**

```bash [install.sh]
$ curl -fsSL https://pget.tempo.xyz/install.sh | bash
```

**From source:**

```bash [build-from-source.sh]
$ git clone https://github.com/tempoxyz/pget.git
$ cd pget
$ cargo install --path .
```

## Quick start

```bash [quickstart.sh]
# Log in (first time only)
$ pget login

# Make a paid request
$ pget query https://mpp.tempo.xyz/api/ping/paid
```

:::tip[Live demo server]
We host a live demo server at `mpp.tempo.xyz` running on **Tempo testnet**. Use `/api/ping` for free requests and `/api/ping/paid` for paid requests. Get testnet funds at [faucet.tempo.xyz](https://faucet.tempo.xyz).
:::

## Configuration

`pget` uses encrypted keystores for secure wallet management.

### Example config

```toml
[evm]
keystore = "/Users/you/.pget/keystores/my-wallet.json"

# Override RPC URLs
[rpc]
tempo = "https://my-private-rpc.com"
```

## Key management

```bash [key-management.sh]
# List all access keys
$ pget keys list

# Switch to a different access key
$ pget keys switch 1

# Delete an access key
$ pget keys delete 2
```

## Command reference

| Command | Description |
|---------|-------------|
| `pget query <url>` | Make an HTTP request with automatic payment |
| `pget login` | Log in and set up your wallet |
| `pget logout` | Log out and clear credentials |
| `pget whoami` | Show the current authenticated identity |
| `pget inspect <url>` | Inspect payment requirements |
| `pget balance` | Check wallet balance |
| `pget config` | View/manage configuration |
| `pget networks` | List supported networks |

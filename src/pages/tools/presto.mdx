# presto [Command-line HTTP client for MPP]

`presto` is a non-interactive command-line tool for making HTTP requests with automatic payment support.

When you request a resource that requires payment, `presto` automatically detects the `402` response, fulfills the payment, and retries—all in a single command.

## Installation

**Quick install:**

```bash [install.sh]
$ curl -fsSL https://presto-binaries.tempo.xyz/install.sh | bash
```

**From source:**

```bash [build-from-source.sh]
$ git clone https://github.com/tempoxyz/presto.git
$ cd presto
$ cargo install --path .
```

## Quick start

```bash [quickstart.sh]
# Log in (first time only — opens browser to authenticate with your Tempo wallet)
$ presto login

# Make a paid request
$ presto https://openrouter.mpp.tempo.xyz/v1/chat/completions \
  -X POST --json '{"model":"openai/gpt-4o-mini","messages":[{"role":"user","content":"Hello!"}]}'
```

:::tip[Live demo server]
We host a live demo server at `mpp.tempo.xyz` running on **Tempo testnet**. Use `/api/ping` for free requests and `/api/ping/paid` for paid requests. Get testnet funds at [faucet.tempo.xyz](https://faucet.tempo.xyz).
:::

## Configuration

`presto login` authenticates via your browser and stores wallet credentials securely.

### File locations

| Platform | Config | Wallet |
|----------|--------|--------|
| **macOS** | `~/Library/Application Support/presto/config.toml` | `~/Library/Application Support/presto/keys.toml` |
| **Linux** | `~/.config/presto/config.toml` | `~/.local/share/presto/keys.toml` |

### Example config

```toml
# RPC overrides for built-in networks
tempo_rpc = "https://my-custom-tempo-rpc.com"
moderato_rpc = "https://my-custom-moderato-rpc.com"
```

## Command reference

| Command | Description |
|---------|-------------|
| `presto <url>` | Make an HTTP request with automatic payment |
| `presto login` | Log in to your Tempo wallet |
| `presto logout [--yes]` | Log out and disconnect your wallet |
| `presto whoami` | Show wallet address, balances, and access keys |
| `presto session list` | List active payment sessions |
| `presto session list --all` | Show all channels: active, orphaned, and closing |
| `presto session list --orphaned` | Scan on-chain for orphaned channels (no local session) |
| `presto session list --closed` | Show channels pending finalization |
| `presto session close [url]` | Close a payment session by URL or channel ID |
| `presto session close --all` | Close all active sessions and on-chain channels |
| `presto session close --orphaned` | Close only orphaned on-chain channels |
| `presto session close --closed` | Finalize channels pending close (grace period elapsed) |
| `presto completions <shell>` | Generate shell completions (bash, zsh, fish, powershell) |

### Global options

| Flag | Description |
|------|-------------|
| `-c, --config <PATH>` | Configuration file path |
| `--key <NAME>` | Use a specific access key |
| `--private-key <KEY>` | Use a private key directly for payment (bypasses wallet login) |
| `-n, --network <NETWORKS>` | Filter to specific networks (comma-separated) |
| `-v, --verbose` | Verbosity level (repeatable: `-v`, `-vv`, `-vvv`) |
| `-q, --quiet` | Suppress log messages |
| `--color <MODE>` | Color output: `auto`, `always`, `never` (default: `auto`) |
| `--output-format <FORMAT>` | Output format: `text`, `json` (default: `text`) |

### Query options

| Flag | Description |
|------|-------------|
| `--dry-run` | Preview payment without executing |
| `-i, --include` | Include HTTP response headers in output |
| `-o, --output <FILE>` | Write response to file |
| `-X, --request <METHOD>` | HTTP method (GET, POST, etc.) |
| `-H, --header <HEADER>` | Add custom header (repeatable) |
| `--json <DATA>` | Send JSON data (sets Content-Type automatically) |
| `-d, --data <DATA>` | POST data (use `@filename` to read from file, `@-` for stdin) |
| `--no-redirect` | Disable following redirects |
| `-m, --timeout <SECONDS>` | Request timeout in seconds |

### Environment variables

| Variable | Description |
|----------|-------------|
| `PRESTO_RPC_URL` | Override RPC endpoint for all networks |
| `PRESTO_PRIVATE_KEY` | Provide a private key directly for payment (bypasses wallet login) |
| `PRESTO_AUTH_URL` | Override auth server URL |
| `PRESTO_NO_TELEMETRY` | Disable telemetry |

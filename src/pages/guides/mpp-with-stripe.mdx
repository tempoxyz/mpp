# Accept MPP with Stripe [Charge per request to your API using Stripe]

Monetize your API with Stripe. Charge agents per request for data, services, or compute.

- **Payments**: Payments land directly in your Stripe balance and settle in fiat.
- **Refunds**: Refunds are available through the API and in the Stripe Dashboard.
- **Payouts**: Metrics, reporting, and multi-currency payouts work the same as any other payment in Stripe.

To learn more, read the [Stripe documentation](https://docs.stripe.com/payments/machine/mpp) on accepting MPP.

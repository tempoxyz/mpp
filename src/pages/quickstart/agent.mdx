# Use with your Agent [Interact with MPP services with your agent]

Three tools for making paid HTTP requests. Pick the one that fits your workflow.

| Tool | Best for | Install |
|------|----------|---------|
| [Tempo Wallet](https://cli.tempo.xyz) | AI coding agents (Amp, Claude Code) | `curl -fsSL https://cli.tempo.xyz/install \| bash` |
| [mppx](https://mpp.dev/sdk/typescript/client/Mppx.create) | JS/TS apps and scripts | `npm install mppx viem` |
| [purl](https://purl.dev) | Humans in the terminal | `cargo install --path cli` |

## Tempo Wallet

A managed wallet CLI with passkey auth, spending limits, and automatic payment. Designed for AI agents that need to make paid requests without manual intervention.

[Docs →](https://cli.tempo.xyz) · [GitHub →](https://github.com/tempoxyz/wallet)

:::::steps

### Install

```bash
curl -fsSL https://cli.tempo.xyz/install | bash
tempo wallet login
```

### Discover services

Prompt your agent to discover what services are available.

::::code-group


## mppx

A TypeScript library that polyfills `fetch` to handle `402` responses automatically. Your existing code works unchanged — payments happen in the background.

[Docs →](https://mpp.dev/sdk/typescript/client/Mppx.create) · [GitHub →](https://github.com/wevm/mppx)

:::::steps

### Install

::::code-group
```bash [npm]
npm install mppx viem
```
```bash [pnpm]
pnpm add mppx viem
```
```bash [bun]
bun add mppx viem
```
::::

### Create payment handler

```ts
import { privateKeyToAccount } from 'viem/accounts'
import { Mppx, tempo } from 'mppx/client'

const account = privateKeyToAccount('0xabc…123')

Mppx.create({
  methods: [tempo({ account })],
})
```

### Make a paid request

```ts
const response = await fetch('https://api.example.com/data')
```

`fetch` is polyfilled — when the server returns `402`, `mppx` pays and retries automatically.

## purl

A curl-like CLI for making HTTP requests that require payment. Built by Stripe for humans and agents alike.

[Docs →](https://purl.dev) · [GitHub →](https://github.com/stripe/purl)

:::::steps

### Install

```bash
git clone https://github.com/stripe/purl
cd purl
cargo install --path cli
```

### Set up a wallet

```bash
purl wallet add
```

### Make a paid request

```bash
# Preview payment without executing
purl --dry-run https://api.example.com/data

# Make the request (pays automatically)
purl https://api.example.com/data

# Require confirmation before paying
purl --confirm https://api.example.com/data
```

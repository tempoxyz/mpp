# Use with your agent [Interact with MPP enabled services with your agent of choice]

Agents and CLIs can pay for MPP-enabled services automatically. Here's how to get started.

## Tempo Wallet <Badge variant="info">Recommended</Badge>

The [Tempo Wallet](https://wallet.tempo.xyz) CLI gives your agent a managed wallet with passkey auth, spending limits, and automatic payment.

[Docs →](https://cli.tempo.xyz) · [GitHub →](https://github.com/tempoxyz/wallet)

### Use in your agent

Paste this into your coding agent to set up Tempo Wallet:

> Read and follow https://wallet.tempo.xyz/skill.md

The agent will install the CLI, connect your wallet, and load the skill automatically.

### Manual install

```bash
# Install the CLI (also installs agent skills automatically)
curl -fsSL https://cli.tempo.xyz/install | bash

# Connect your wallet (opens browser for passkey auth)
tempo wallet login

# Verify setup
tempo wallet whoami
```

Once installed, discover and call services:

```bash
# List available services
tempo mpp services

# Make a paid request
tempo mpp -X POST \
  --json '{"prompt": "a sunset over the ocean"}' \
  https://fal.mpp.tempo.xyz/fal-ai/flux/dev
```

## mppx

The [`mppx`](/sdk/typescript/cli) CLI is a lightweight alternative bundled with the `mppx` npm package.

[Docs →](/sdk/typescript/cli) · [GitHub →](https://github.com/wevm/mppx)

```bash
# Create an account
npx mppx account create

# Make a paid request
npx mppx https://mpp.dev/api/ping/paid
```

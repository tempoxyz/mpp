# Use with your Agent [Interact with MPP services with your agent]

The Machine Payments Protocol (MPP) can be used with agents to discover and use [services](/services) via direct integrations with MPP-compatible command-line interfaces (CLIs).

## Tempo Wallet

Tempo Wallet provides a first-class integration with MPP to provide a seamless experience for using services with your preferred agent.

:::::steps

### Install

First, install the `tempo` binary via your preferred terminal.

```bash [install.sh]
curl -L cli.tempo.xyz/install | bash
```

### Prompt agent & discover services

You can discover what services are available by prompting your agent.

::::code-group

```bash [Claude]
claude prompt "What image generation services can I use with Tempo Wallet?"
```

```bash [Amp]
echo "What image generation services can I use with Tempo Wallet?" | amp
```

```bash [Codex]
codex "What image generation services can I use with Tempo Wallet?"
```

```bash [Other]
# Prompt your agent
> "What image generation services can I use with Tempo Wallet?"
```

::::

### Perform services

Prompt your agent to use a service. Presto and MPP handle payments on the fly.

::::code-group

```bash [Claude]
claude "Generate a surreal image with fal.ai"
```

```bash [Amp]
echo "Generate a surreal image with fal.ai" | amp
```

```bash [Codex]
codex "Generate a surreal image with fal.ai"
```

```bash [Other]
# Prompt your agent
> "Generate a surreal image with fal.ai"
```

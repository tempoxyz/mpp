# Use with your agent [Interact with MPP enabled services with your agent of choice]

Agents and CLIs can pay for MPP-enabled services automatically. Here's how to get started.

| Tool | Best for | Install |
|------|----------|---------|
| [Tempo Wallet](https://wallet.tempo.xyz) | AI coding agents (Amp, Claude Code, Codex) | `curl -fsSL https://cli.tempo.xyz/install \| bash` |
| [mppx](https://mpp.dev/sdk/typescript/cli#cli-reference) | Basic validation and scripts | `npm install mppx viem` |
| [purl](https://purl.dev) | Humans in the terminal | `cargo install --path cli` |

## mppx

The [`mppx`](/sdk/typescript/cli#cli-reference) CLI provides a lightweight command-line and scripting interface for making requests to MPP-enabled services.

[Docs →](/sdk/typescript/cli) · [GitHub →](https://github.com/wevm/mppx)

:::::steps

### Install

::::code-group
```bash [npm]
npm install -g mppx
```
```bash [pnpm]
pnpm add -g mppx
```
```bash [bun]
bun add -g mppx
```
::::

### Make a paid request

```bash
mppx https://api.example.com/data
```

Payment is handled automatically when the server returns `402`.

### Pass method options

```bash
mppx https://api.example.com/data -M channel=0x123 -M deposit=1000000
```

## purl

A curl-like CLI for making HTTP requests that require payment. Built by Stripe for humans and agents alike.

[Docs →](https://purl.dev) · [GitHub →](https://github.com/stripe/purl)

:::::steps

### Install

```bash
git clone https://github.com/stripe/purl
cd purl
cargo install --path cli
```

### Set up a wallet

```bash
purl wallet add
```

### Make a paid request

```bash
# Preview payment without executing
purl --dry-run https://api.example.com/data

# Make the request (pays automatically)
purl https://api.example.com/data

# Require confirmation before paying
purl --confirm https://api.example.com/data
```

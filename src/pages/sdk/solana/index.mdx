# Solana SDK [Solana payments for MPP]

## Overview

The `mppx-solana` package adds Solana payment methods to `mppx` so you can accept SOL and SPL token payments in the standard MPP Challenge, Credential, and Receipt flow.

## Install

```bash [install.sh]
$ pnpm add mppx-solana mppx @solana/spl-token @solana/web3.js
```

## Package details

- **Package:** `mppx-solana`
- **Repository:** `nitishxyz/mppx-solana`
- **Works with:** `mppx`
- **Supports:** SOL and SPL token payments

## What you can build

- Protect HTTP endpoints with Solana-based payment Challenges
- Accept native SOL payments
- Accept SPL token payments such as USDC
- Verify transactions on-chain before you return a Receipt

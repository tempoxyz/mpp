# `Intent.session` [Pay-as-you-go payment sessions]

Session intent for pay-as-you-go payment sessions using cumulative vouchers over a payment channel.

## Usage

```ts twoslash
import { Intent } from 'mppx'

const session = Intent.session
```

## Type

```ts
const session: Intent
```

## Schema

### request

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `amount` | `string` | ✓ | Amount to charge per unit |
| `unitType` | `string` | ✓ | Unit type label (for example, "token", "byte", "request") |
| `currency` | `string` | ✓ | Currency identifier |
| `decimals` | `number` | ✓ | Decimal places for amount parsing |
| `recipient` | `string` | | Payment recipient |
| `suggestedDeposit` | `string` | | Suggested deposit amount for clients |

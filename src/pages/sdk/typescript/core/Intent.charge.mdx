# `Intent.charge` [One-time immediate payments]

Charge intent for one-time immediate payments.

## Usage

```ts twoslash
import { Intent } from 'mpay'

// Use the built-in charge intent
const charge = Intent.charge
```

## Type

```ts
const charge: Intent
```

## Schema

### request

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `amount` | `string` | ✓ | Amount to charge |
| `currency` | `string` | ✓ | Currency identifier |
| `description` | `string` | | Human-readable description |
| `expires` | `string` | | Expiration timestamp (ISO 8601). Defaults to 5 minutes. |
| `externalId` | `string` | | External reference ID |
| `recipient` | `string` | | Payment recipient |

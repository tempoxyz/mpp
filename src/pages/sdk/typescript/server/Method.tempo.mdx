# `tempo` [Tempo payment method namespace]

The `tempo` namespace provides server-side payment methods for the Tempo blockchain.

## Methods

| Method | Description |
|--------|-------------|
| [`tempo.charge()`](/sdk/typescript/server/Method.tempo.charge) | One-time payments |
| [`tempo.stream()`](/sdk/typescript/server/Method.tempo.stream) | Low-cost high-throughput payments |

## Usage

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  methods: [tempo.charge()],
})
```

### With multiple intents

Register both charge and stream methods on a single server.

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  // [!code focus:start]
  methods: [
    tempo.charge(),
    tempo.stream({ rpcUrl: 'https://rpc.tempo.xyz', storage: {} as any }),
  ],
  // [!code focus:end]
})
```

## Shared parameters

Both `tempo.charge()` and `tempo.stream()` accept these common parameters:

### chainId (optional)

- **Type:** `number`

Chain ID for the Tempo network.

### rpcUrl (optional)

- **Type:** `{ [chainId: number]: string }` (charge) or `string` (stream)

RPC URL(s) for connecting to the Tempo blockchain.

### testnet (optional)

- **Type:** `boolean`

Testnet mode. When `true`, uses the Tempo testnet chain ID.

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  methods: [tempo.charge({
    testnet: true, // [!code focus]
  })],
})
```

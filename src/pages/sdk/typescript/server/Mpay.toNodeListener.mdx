# `Mpay.toNodeListener`

Wraps a payment handler to create a Node.js HTTP listener.

## Usage

```ts twoslash
// @noErrors
import * as http from 'node:http'
import { Mpay, tempo } from 'mpay/server'

const payment = Mpay.create({
  method: tempo(),
})

http.createServer(async (req, res) => {
  const result = await Mpay.toNodeListener(
    payment.charge({
      amount: '1000', 
      currency: '0x...', 
      recipient: '0x...',
    }),
  )(req, res)

  if (result.status === 402) return

  res.end('OK')
})
```

## Behavior

- **On `402`:** Writes the challenge response headers and body, then ends the connection.
- **On `200`:** Sets the `Payment-Receipt` header; caller should write the response body.

## Return Type

```ts
type ReturnType = (
  req: IncomingMessage,
  res: ServerResponse,
) => Promise<IntentFn.Response<Transport.Http>>
```

## Parameters

### handler

- **Type:** `(input: Request) => Promise<IntentFn.Response<Transport.Http>>`

The payment handler function returned by calling an intent on the payment object.

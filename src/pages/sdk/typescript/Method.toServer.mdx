# `Method.toServer`

Extends a method with server-side verification logic.

## Usage

```ts twoslash
import { Intent, Method, MethodIntent, Receipt, z } from 'mpay'

const charge = MethodIntent.fromIntent(Intent.charge, {
  method: 'stripe',
  schema: {
    credential: {
      payload: z.object({ signature: z.string() }),
    },
  },
})

const stripe = Method.from({
  name: 'stripe',
  intents: { charge },
})

declare function verifySignature(credential: unknown): Promise<boolean>
// ---cut---
const server = Method.toServer(stripe, {
  async verify({ credential }) {
    const isValid = await verifySignature(credential)
    
    return Receipt.from({
      method: 'stripe',
      status: 'success',
      timestamp: new Date().toISOString(),
      reference: '0x...',
    })
  },
})
```

## Return Type

```ts
type ReturnType = Server<name, intents>
```

A server-side payment method with verification logic.

## Parameters

### method

- **Type:** `Method`

The base method to extend.

### options

#### request (optional)

- **Type:** `(parameters) => request`

Transform request before challenge creation.

#### verify

- **Type:** `(parameters: { credential, request }) => Promise<Receipt>`

Function that validates a credential and returns a receipt.

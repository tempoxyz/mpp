# Next.js [Payment middleware for Next.js]

Native [Next.js](https://nextjs.org) route handler wrapper that gates routes behind payment intents.

## Install

:::code-group

```bash [npm]
npm install mpay
```

```bash [pnpm]
pnpm add mpay
```

```bash [bun]
bun add mpay
```

:::

## Usage

Import `Mpay` and `tempo` from `mpay/nextjs` to create a Next.js-aware payment handler. Each intent (for example, `charge`) returns a wrapper that accepts a route handler.

```ts [app/api/premium/route.ts]
import { Mpay, tempo } from 'mpay/nextjs'

const mpay = Mpay.create({ methods: [tempo.charge()] }) // [!code hl]

export const GET = 
  mpay.charge({ amount: '1' }) // [!code hl]
  (() => Response.json({ data: 'paid content' }))
```

# `tempo`

Creates a Tempo payment method client for signing payment transactions.

## Usage

```ts twoslash
// @noErrors
import { Fetch, tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const account = privateKeyToAccount('0x...')

const fetch = Fetch.from({
  methods: [
    // [!code focus:start]
    tempo({ account }),
    // [!code focus:end]
  ],
})
```

## Return Type

```ts
type Tempo = Method.Client<
  'tempo',
  Schema,
  z.ZodObject<{ account: z.ZodOptional<z.ZodCustom<Account>> }>
>
```

## Parameters

### account (optional)

- **Type:** `Account`

Account to sign transactions with. Can be overridden per-call via context.

```ts twoslash
// @noErrors
import { tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const method = tempo({
  account: privateKeyToAccount('0x...'), // [!code focus]
})
```

### rpcUrl (optional)

- **Type:** `{ [chainId: number]: string }`

RPC URLs keyed by chain ID.

```ts twoslash
// @noErrors
import { tempo } from 'mpay/client'

const method = tempo({
  // [!code focus:start]
  rpcUrl: {
    123456: 'https://rpc.tempo.xyz',
  },
  // [!code focus:end]
})
```

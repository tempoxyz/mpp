# `Transport.http`

HTTP transport for client-side payment handling.

## Usage

```ts twoslash
// @noErrors
import { Mpay, Transport, tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const account = privateKeyToAccount('0x...')

const mpay = Mpay.create({
  methods: [tempo({ account })],
  transport: Transport.http(), // [!code focus]
})
```

## Behavior

- Detects payment required via **`402` status code**
- Extracts challenges from **`WWW-Authenticate` header**
- Sends credentials via **`Authorization` header**

## Return Type

```ts
type HttpTransport = Transport<RequestInit, Response>
```

## Parameters

None.

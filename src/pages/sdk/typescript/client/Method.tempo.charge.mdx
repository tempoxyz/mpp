# `Method.tempo.charge` [One-time payments]

Creates a Tempo charge payment method for client-side transaction signing.

## Usage

```ts twoslash
// @noErrors
import { Fetch, tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const account = privateKeyToAccount('0x...')

const fetch = Fetch.from({
  methods: [
    // [!code focus:start]
    tempo.charge({ account }),
    // [!code focus:end]
  ],
})

const response = await fetch('https://mpp.dev/api/ping/paid')
```

### With polyfill

```ts twoslash
// @noErrors
import { Fetch, tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const account = privateKeyToAccount('0x...')

Fetch.polyfill({
  methods: [tempo.charge({ account })],
})
```

## Return type

```ts
import type { MethodIntent } from 'mpay'

type ReturnType = MethodIntent.Client
```

## Parameters

### account (optional)

- **Type:** `Account`

Account to sign transactions with. You can override this per call using the context.

```ts twoslash
// @noErrors
import { tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const method = tempo.charge({
  account: privateKeyToAccount('0x...'), // [!code focus]
})
```

### getClient (optional)

- **Type:** `(chainId: number) => Client`

Function that returns a viem client for the given chain ID.

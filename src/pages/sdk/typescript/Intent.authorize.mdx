# `Intent.authorize` [Pre-authorization with spending limits]

Authorize intent for pre-authorization with spending limits.

## Usage

```ts twoslash
import { Intent } from 'mpay'

// Use the built-in authorize intent
const authorize = Intent.authorize
```

## Type

```ts
const authorize: Intent
```

## Schema

### request

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `amount` | `string` | ✓ | Amount to authorize |
| `currency` | `string` | ✓ | Currency identifier |
| `expires` | `string` | ✓ | Expiration timestamp (ISO 8601) |
| `description` | `string` | | Human-readable description |
| `externalId` | `string` | | External reference ID |
| `recipient` | `string` | | Payment recipient |

## Reference

See the [authorize intent specification](https://github.com/tempoxyz/payment-auth-spec/blob/main/specs/intents/draft-payment-intent-authorize-00.md).

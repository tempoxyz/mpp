# `MethodIntent.fromIntent`

Creates a method-specific intent from a base intent.

## Usage

```ts twoslash
import { Intent, MethodIntent, z } from 'mppx'

const tempoCharge = MethodIntent.fromIntent(Intent.charge, {
  method: 'tempo',
  schema: {
    credential: {
      payload: z.object({
        signature: z.string(),
        type: z.literal('transaction'),
      }),
    },
    request: {
      methodDetails: z.object({
        chainId: z.number(),
      }),
      requires: ['recipient'],
    },
  },
})
```

## Return type

```ts
type ReturnType = MethodIntent<intent, options>
```

A method-specific intent with merged request schema.

## Parameters

### intent

- **Type:** `Intent`

The base intent to extend (for example, `Intent.charge`).

### options

#### method

- **Type:** `string`

Payment method name (for example, "tempo", "stripe").

#### schema

##### credential.payload

- **Type:** `z.ZodMiniType`

Zod schema for the credential payload.

##### request (optional)

###### methodDetails (optional)

- **Type:** `z.ZodMiniObject`

Additional method-specific fields to add to the request schema. These fields are nested under `methodDetails` in the output.

###### requires (optional)

- **Type:** `ReadonlyArray<keyof intent['schema']['request']>`

Array of field names from the base intent that should be required (unwraps optional fields).
